# Gym Performance Dashboard

## 📊 Project Overview
This project analyzes gym member data using SQL Server and Power BI.

## 🔧 Tools Used
- SQL Server
- Power BI

## 📈 Key Insights
- HIIT burns the highest calories.
- More experienced users burn more calories.
- Higher activity level leads to higher calorie burn.
